はじめに…
扇風機を用いたシュミレーションゲームとなっております。
開発目的として、実際にエコシステムを利用したゲームなどはあったのですが、扇風機単体をゲームしたらどのような面白さになるかを考えました。
やり方
難易度を選択し、時間内に、扇風機使用率を60％使用し、一定時間に出来ればクリアとなります。
操作方法はマウスのみでできます。

工夫した所
難易度を設定する際に、どのくらいの時間に設定すればよいのかを悩みました。
簡単にしすぎるとクリアしやすいため、バランス維持に主に時間を掛けました。

使用言語
C++/DxLibライブラリ
